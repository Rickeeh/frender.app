What it does

  • Turns audio files into MP4 using a single static cover image.

  • Exports consistently (1 FPS, CFR), ready to publish.

  • Batch workflows for albums, EPs, and episodes.

What it doesn't do

  • Not a video editor.

  • No animations. No motion.

  • No technical knobs — codecs, fps, or bitrates are handled automatically.

  • No cloud dependency. Everything runs offline.

Under the hood

Deterministic render pipeline

Same input always produces the same output. Vendored FFmpeg, pinned SHA-256.

Real cancel with full cleanup

Cancellation kills the process tree and removes all partial files. No leftovers.

Crash-safe recovery

A job ledger tracks in-progress exports. Interrupted jobs are cleaned up on next launch.
